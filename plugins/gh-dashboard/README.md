# gh-dashboard

GitHub project status dashboard for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| `project-status` | Fetches and displays GitHub project status — board, milestones, PRs, activity |

## Usage

```
gh-dashboard:project-status
```

Or use natural language: "What's the status of the project?", "Show me open PRs", "What should I work on?"

## Installation

```bash
claude plugins marketplace add ImCesar/cezaar
claude plugins install gh-dashboard
```
